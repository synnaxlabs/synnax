Tools and Technologies

The following is a list of major tools and technologies we used to develop Synnax. Some of the more obvious ones are emitted (e.g. HTML, CSS, TypeScript, Go etc.) for brevity. The list is in alphabetical order.

Chromatic​

Chromatic is a visual regression testing tool used alongside Storybook to ensure that the UI components we develop are consistent. We currently uses it for versioning our [Pluto](https://github.com/synnaxlabs/synnax/tree/main/pluto) components.

Codecov​

Codecov is a code coverage tool used to ensure that our code is sufficiently tested.

Docker and Docker Hub​

Container images are the best way to package and deploy Synnax across many environments. We currently push all of our images to docker hub.

Docusaurus​

This very site is built using Facebook's Docusaurus. It turns our Markdown and MDX files into beautiful, syntax highlighted, and well organized documentation.

Earthly​

Earthly is a build automation tool that runs all of its commands in a container. This allows us to build and test our code in a consistent environment. We use it for makefile style utilities as well as CI based builds in Github Actions.

Ginkgo and Gomega​

Ginkgo is a BDD style testing framework that we use for all of our go tests. It maintains a similar API to Javascript's Jest or AVA. Ginko is paired with Gomega, a matcher library that allows us to write assertions.

Github, Github Actions, and Github Container Registry​

Github is the home of all of our code. We use GitHub actions for all our our CI/CD needs, including pull linting, pull request testing, and release automation. We store all of our container images images in Github's container registry as well as Docker Hub.

gRPC​

gRPC is a high performance, RPC framework that we use for all cluster-internal communication as well as Synnax's client facing API.

Pebble​

CockroachDB's Pebble is a go-embedded key-value store that we use for persisting all of Synnax's meta data to disk.

Protocol Buffers​

Protocol Buffers are a language and platform neutral serialization format that we use to define all of our gRPC API's.

Pytest​

Pytest is a testing framework for Python that we use for all of our Python tests.

React​

React is a Javascript library that we use for all of our frontend components.

Storybook​

Storybook is a UI development environment that we use to develop and test our React components. It allows us to develop components in isolation and view them in a variety of different states.

Vite​

Vite is a Javascript bundler that we use to build our frontend code. It is similar to Webpack but is much faster and has a much smaller bundle size.

Yarn​

Yarn is a Javascript package manager that we use to manage all of our frontend dependencies.

Websockets​

Websockets are a protocol that we use to stream telemetry from Synnax's API to the UI as well as to some client libraries
