In this series of guides, we'll walk you through all you need to productively instrument
and operate hardware using Synnax.

## Outline

We'll walk you through the following steps:

1. Install the Synnax Console and Cluster.
2. Acquire data and visualize sensor data from a device.
3. Use schematics to manually control actuators.
4. Create control sequences to automate operation.
5. Review, organize, and annotate data with ranges.
6. Set up workspaces to configure repeatable layouts.
