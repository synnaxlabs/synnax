This page will guide you through the basics of installing and using the Synnax visualization
UI, Delta.

## Prerequisites

None.

## Overview

The Synnax UI allows you to manage a cluster and visualize its telemetry. Use it to:

1. Visualize channel data.
2. Filter relevant data for post processing using a [client library.](/analyze/get-started)
3. Observe cluster metrics and system health.
4. Configure channels and manage cluster resources.

## Install

Install Delta by downloading the latest release for your operating system.
