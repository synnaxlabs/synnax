## Examples

Many of [our examples](/reference/python-client/examples) show how to effectively read
data from a Synnax Core. Some examples of note are:

- [Plot](https://github.com/synnaxlabs/synnax/tree/main/client/py/examples/plot.py)
  - Demonstrates how to plot data retrieved from a Synnax Core.
- [Export to CSV](https://github.com/synnaxlabs/synnax/tree/main/client/py/examples/export_to_csv.py)
  - Demonstrates how to export data retrieved from a Synnax Core to a CSV file.
- [Read Latest](https://github.com/synnaxlabs/synnax/tree/main/client/py/examples/read_latest.py)
  - Demonstrates how to read the latest x samples from a channel.
