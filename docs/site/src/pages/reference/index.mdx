Synnax is a sensor data platform that handles everything from collecting measurements
off your hardware to storing, visualizing, and acting on them in real time. Connect your
devices, build dashboards, and write control sequences, all from a single tool.

Whether you're running a test stand, commissioning a new system, or automating a
production line, [get started by installing Synnax](/reference/installation).
