The clients support several different ways of reading data from a Core. We can
read directly from a channel, fetch a range and access its data, or leverage server-side
iterators for processing large queries.

If you'd like a conceptual overview of how to read data in Synnax, check out the
[reads](/reference/concepts/reads) page.
