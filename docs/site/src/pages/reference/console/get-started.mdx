The Synnax Console is a desktop application for visualizing data, controlling hardware,
and managing a Synnax cluster. This page will get you started with installing the
console, and point you to the right resources for learning how to use it.

<Divider.Divider x />

Install the console by downloading the latest release for your operating system:
