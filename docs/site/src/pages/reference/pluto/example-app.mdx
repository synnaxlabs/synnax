We've put together a simple typescript app built with [Vite](https://vitejs.dev/) and
Pluto. It's available in the
[`pluto/examples`](https://github.com/synnaxlabs/synnax/tree/main/pluto/examples)
directory of the [Synnax](https://github.com/synnaxlabs/synnax) repository. In this
guide, we'll show you how to clone the repo and run the app locally.

## Clone the Synnax Repository

First, clone the Synnax repository to your local machine:

```bash
git clone https://github.com/synnaxlabs/synnax
```

## Install Dependencies

Then, move into the `pluto/examples` directory and install the dependencies:

```bash
cd synnax/pluto/examples && npm install
```

## Run the App

Finally, run the app:

```bash
npm run dev
```

Then, navigate to `http://localhost:5173` in your browser.
