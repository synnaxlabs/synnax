Synnax offers several device drivers to help you connect your data acquisition and
control hardware out of the box.

<Divider.Divider direction="x" />

## Available Drivers

| Component                                                        | Description                               |
| ---------------------------------------------------------------- | ----------------------------------------- |
| [OPC UA](/reference/device-drivers/opc-ua/connect-server)        | Connect to OPC UA servers.                |
| [National Instruments](/reference/device-drivers/ni/get-started) | Connect to National Instruments hardware. |
| [LabJack](/reference/device-drivers/labjack/get-started)         | Connect to LabJack hardware.              |

## Building a Custom Driver

If you don't see a driver for your hardware, you can build your own using the [Python
client](/reference/python-client/device-driver).
