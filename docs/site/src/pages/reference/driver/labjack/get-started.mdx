The Synnax Driver supports reading and writing data from LabJack devices. This page will
walk you through the steps to start acquiring data from your LabJack devices using the
Driver.

<Divider.Divider x />

## Installing LJM

<Text.Text level="p">
  <Icon.Logo.LabJack /> [LabJack LJM library](https://support.labjack.com/docs/ljm-software-installer-downloads-t4-t7-t8-digit) must installed for the Driver to communicate with your LabJack devices.

</Text.Text>

<Divider.Divider x />

## Supported Devices

Synnax supports reading and writing data to all LabJack T-Series devices.

<Note.Note variant="warning" className="compact">
  Currently, only T-Series LabJack devices are supported by Synnax. If you are
  interested in using Synnax with other devices, please [contact
  us](https://www.synnaxlabs.com/contact).
</Note.Note>
