the server.

<Divider.Divider x />

## Prerequisites

Before you can connect to a Modbus TCP server, you will need to ensure that you have the
following:

1. A [Synnax Core](/reference/core/quick-start) running on your network.
2. A [Synnax Driver](/reference/driver/get-started) running on your network. This Driver
   must be connected to the Core and running on the same machine or network as your
   Modbus server.
3. The [Synnax Console](/reference/console/get-started) installed on your local machine
   (for Console method).
4. A Modbus TCP server running on your network. This server must be reachable by the
   Driver.

## <StepText level="h2" step={1}>Open the Connection Dialog</StepText>

To connect to your Modbus server, open the Console and click on the Search and Command
Palette at the top. You can also open this palette with `Ctrl+Shift+P` on Windows or
`Cmd+Shift+P` on macOS.

In command mode (enabled when the first character in the input is `>`), type "Modbus".
Select the "Connect a Modbus Server" option to open the server configuration dialog.

<ExampleDetails open>
<Text.Text slot="summary" level="p"><Icon.Connect /> Connect to Modbus Server</Text.Text>

```python
import synnax as sy
from synnax.hardware import modbus
import json

# Initialize Synnax client
client = sy.Synnax()

# Retrieve the embedded rack
rack = client.hardware.racks.retrieve_embedded_rack()

# Create the Modbus device
device = modbus.create_device(
    client=client,
    name="My Modbus Server",
    location="192.168.1.100:502",
    rack=rack.key,
    properties=json.dumps(
        modbus.device_props(
            host="192.168.1.100",
            port=502,
            swap_bytes=False,
            swap_words=False,
        )
    ),
)

print(f"Connected to Modbus server: {device.name}")
```

</ExampleDetails>

<ExampleDetails open>
<Text.Text slot="summary" level="p"><Icon.Close /> Disconnect Server</Text.Text>

```python
import synnax as sy

# Initialize Synnax client
client = sy.Synnax()

# Retrieve the device by name
device = client.hardware.devices.retrieve(
    name="My Modbus Server",
    ignore_not_found=True
)

if device is not None:
    # Delete the server
    client.hardware.devices.delete([device.key])
    print(f"Disconnected Modbus server: {device.name}")
else:
    print("Server not found")
```
