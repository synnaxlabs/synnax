Acquires data from digital inputs on NI devices at a specified sample rate.

<Divider.Divider x />

## Task Configuration Reference

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `name` | string | Yes | - | Human-readable task name |
| `sample_rate` | number | Yes | - | Samples per second (Hz) |
| `stream_rate` | number | No | `sample_rate` | Rate data is streamed to Synnax (Hz), must be â‰¤ `sample_rate` |
| `data_saving` | boolean | No | `false` | Enable permanent storage in Synnax |
| `auto_start` | boolean | No | `false` | Automatically start task after configuration |
| `channels` | array | Yes | - | List of digital input channel configurations |

<Divider.Divider x />

## Channel Configuration
[DI Channel NI-DAQmx C API Reference](https://www.ni.com/docs/en-US/bundle/ni-daqmx-c-api-ref/page/daqmxcfunc/daqmxcreatedichan.html)

| Parameter | Type | Required | Default | Description |
|-----------|------|----------|---------|-------------|
| `device` | string | Yes | - | Device identifier |
| `port` | number | Yes | - | Physical port |
| `line` | number | Yes | - | Line number on the port |

<Divider.Divider x />

## Important Rules

- **Sample rates**: All channels in a task sample at the same rate. Create separate tasks for different rates.
- **One task per module**: Only one running task can claim a module at a time.
- **Stream rate optimization**: For low-rate tasks (&lt;50Hz), set stream rate = sample rate. For high-rate tasks, keep stream rate &lt;50Hz for better performance.

<ExampleDetails>
<Text.Text slot="summary" level="p"><Icon.Play /> Configure and Run Task</Text.Text>

```python
import synnax as sy
from synnax.hardware import ni

client = sy.Synnax()

# Retrieve device
dev = client.hardware.devices.retrieve(name="my_daq")

# Create index channel
di_time = client.channels.create(
    name="di_time",
    is_index=True,
    data_type=sy.DataType.TIMESTAMP,
    retrieve_if_name_exists=True,
)

# Create data channel
di_0_0 = client.channels.create(
    name="di_0_0",
    index=di_time.key,
    data_type=sy.DataType.UINT8,
    retrieve_if_name_exists=True,
)

# Create and configure task
task = ni.DigitalReadTask(
    name="Digital Read",
    sample_rate=sy.Rate.HZ * 100,
    stream_rate=sy.Rate.HZ * 25,
    data_saving=True,
    channels=[
        ni.DILineChan(
            channel=di_0_0.key,
            device=dev.key,
            port=0,
            line=0,
        ),
    ],
)

client.hardware.tasks.configure(task)

# Start task and read data
with task.run():
    with client.open_streamer(["di_0_0"]) as streamer:
        for _ in range(10):
            frame = streamer.read()
            print(frame)
```

```typescript
import { Synnax } from "@synnaxlabs/client";

const client = new Synnax();

// Retrieve device
const dev = await client.hardware.devices.retrieve({ name: "my_daq" });

// Create index channel
const diTime = await client.channels.create({
  name: "di_time",
  isIndex: true,
  dataType: "timestamp",
  retrieveIfNameExists: true,
});

// Create data channel
const di00 = await client.channels.create({
  name: "di_0_0",
  index: diTime.key,
  dataType: "uint8",
  retrieveIfNameExists: true,
});

// Create and configure task
const task = await client.hardware.tasks.create({
  name: "Digital Read",
  type: "ni_digital_read",
  config: JSON.stringify({
    sample_rate: 100,
    stream_rate: 25,
    data_saving: true,
    channels: [
      {
        type: "di_line",
        channel: di00.key,
        device: dev.key,
        port: 0,
        line: 0,
      },
    ],
  }),
});

// Start task
await task.executeCommandSync("start");

// Read data
const streamer = await client.openStreamer(["di_0_0"]);
for (let i = 0; i < 10; i++) {
  const frame = await streamer.read();
  console.log(frame);
}

// Stop task
await task.executeCommandSync("stop");
await streamer.close();
```

#include "client/cpp/synnax.h"

auto client = synnax::Synnax();

// Retrieve device
auto [dev, dev_err] = client.hardware.retrieve_device("my_daq");
if (dev_err) return dev_err;

// Create index channel
auto di_time = synnax::Channel("di_time", telem::TIMESTAMP_T, true);
auto time_err = client.channels.create(di_time);
if (time_err) return time_err;

// Create data channel
auto di_0_0 = synnax::Channel("di_0_0", telem::UINT8_T, di_time.key, false);
auto data_err = client.channels.create(di_0_0);
if (data_err) return data_err;

// Create and configure task
// (Note: Task configuration in C++ typically done via driver directly)

// Start task and read data
// Open streamer to read incoming data
auto [streamer, stream_err] = client.telem.open_streamer(synnax::StreamerConfig{
    std::vector{di_0_0.key}
});
if (stream_err) return stream_err;

for (int i = 0; i < 10; i++) {
    auto [frame, read_err] = streamer.read();
    if (read_err) return read_err;
    // Process frame data
}

auto close_err = streamer.close();
if (close_err) return close_err;
